# 📖 About

This repository documents my **50-Day RTL Design Challenge**, where I learn digital design by implementing one hardware module every day using **Verilog HDL**.

The focus is not only on writing Verilog but also understanding **how hardware actually works**, verifying designs through simulation, and documenting every concept from scratch.

Each day includes:

- 📚 Theory
- 💻 RTL Code
- 🧪 Testbench
- 📈 GTKWave Simulation
- 📝 Assignment
- 📖 Detailed Documentation
