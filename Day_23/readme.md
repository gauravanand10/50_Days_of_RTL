# Day 23 - Full Range vs Partial Range Counters

## Objective

Learn the difference between **Full Range Counters** and **Partial Range Counters** in Verilog. Understand how binary counters utilize all possible states, while ring and Johnson counters generate only a subset of states for specific applications.

## Design

### Full Range Counter

Inputs:

- Clock (`clk`)
- Reset (`rst`)

Output:

- Counter Output (`count[3:0]`)

Functionality:

- Counts through all possible binary states.
- A 4-bit counter counts from **0000 (0)** to **1111 (15)** before repeating.

### Partial Range Counter

Inputs:

- Clock (`clk`)
- Reset (`rst`)

Output:

- Counter Output (`count[3:0]`)

Functionality:

- Uses only selected states instead of all possible binary combinations.
- Examples include Ring Counter and Johnson Counter.

## Key Observation

During simulation:

- A binary counter visits every possible state.
- A ring counter circulates a single logic '1' through the flip-flops.
- A Johnson counter feeds back the inverted output of the last flip-flop.
- Johnson counters generate twice as many valid states as ring counters.
- Binary counters provide maximum state utilization.

## Assignment

## Key Observation

For every clock cycle:

- Binary Counter counts through all **16** possible states.
- Ring Counter generates only **4** unique states.
- Johnson Counter generates **8** unique states.
- Johnson Counter is more efficient than a Ring Counter for the same number of flip-flops.
- Feedback determines the sequence generated by the counter.

## Comparison: Binary Counter vs Ring Counter vs Johnson Counter

| Feature | Binary Counter | Ring Counter | Johnson Counter |
|---------|----------------|--------------|-----------------|
| Counter Type | Full Range | Partial Range | Partial Range |
| Number of States (4-bit) | 16 | 4 | 8 |
| Feedback | None | Direct Feedback | Inverted Feedback |
| Hardware Complexity | Low | Medium | Medium |
| State Utilization | 100% | 25% | 50% |
| Self Starting | Yes | No | Yes |
| Speed | High | High | High |
| Applications | Timers, Frequency Division | Sequence Generation | FSMs, Sequence Generation, Stepper Motors |

## Conclusion

Full Range Counters utilize every possible binary combination, making them ideal for general-purpose counting applications. Partial Range Counters intentionally limit the sequence to specific states for control and sequencing applications. Among partial range counters, Johnson Counters offer significant advantages over Ring Counters by generating twice as many valid states while remaining self-starting and easy to implement.
